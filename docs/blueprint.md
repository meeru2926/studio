# **App Name**: CakeStory

## Core Features:

- Cinematic Flavor Transformation: An interactive hero system that dynamically shifts themes, color palettes, and cinematic WebP backgrounds between the four flagship cake variants.
- Dynamic Particle Engine: A floating visual system that renders ingredient-specific particles like cocoa dust, strawberry syrup splashes, or sugar crystals using high-performance CSS and GSAP.
- AI Flavor Sommelier: A sophisticated generative tool that uses reasoning to recommend a personalized dessert pairing or flavor journey based on the user's occasion or mood profile.
- Parallax Product Sculptor: Full-screen interactive scroll and hover effects that tilt and rotate hyper-real cake visuals for a high-end tactile feel.
- Variant Persistence State: A state-management system to ensure smooth transitions between variants and cross-section thematic consistency without page reloads.
- Editorial Flavor Carousel: A luxurious masonry grid and scroll-triggered gallery to showcase the micro-textures of high-quality ingredients.
- Pre-flight Asset Orchestrator: A high-end cinematic loader with progress percentages that ensures heavy WebP assets are pre-rendered for an ultra-smooth initial entrance.

## Style Guidelines:

- Primary Color: Gilded Wheat (#D4B35E) — A rich, saturated gold that contrasts with dark environments to signify premium luxury.
- Background Color: Obsidian Roast (#1A1512) — A deeply desaturated, warm-tinted charcoal that provides a theatrical backdrop for hyper-real visuals.
- Accent Color: Toasted Mahogany (#994433) — An analogous warm earthy tone used for active UI elements and delicate highlight transitions.
- Headlines: 'Playfair Display' (Serif) for a high-contrast, editorial fashion aesthetic; Body: 'PT Sans' (Sans-serif) for readable, elegant product descriptions.
- Minimalist stroke icons with 1px weight, featuring subtle glowing borders to match the luxury tech 'Apple-style' controls.
- Cinematic wide-spacing editorial layout with off-center titles, heavy use of negative space, and large-scale interactive floating hero elements.
- High-fidelity parallax motion using smooth easing curves; 'Violent' liquid splashes for chocolate transitions versus soft 'silk' fades for vanilla.

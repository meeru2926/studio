# CakeStory | Luxury Cinematic Desserts

This is a premium, Apple-style product launch application for a luxury cake boutique.

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

## How to Upload to GitHub

To upload this project to your own GitHub repository, follow these standard steps:

1. **Initialize Git:**
   Open your terminal in the project root and run:
   ```bash
   git init
   ```

2. **Add your files:**
   ```bash
   git add .
   ```

3. **Commit your changes:**
   ```bash
   git commit -m "Initial commit of CakeStory Luxury Desserts"
   ```

4. **Create a Repository on GitHub:**
   - Go to [GitHub](https://github.com) and create a new repository.
   - Do **not** initialize it with a README, license, or `.gitignore` (since we already have them).

5. **Connect and Push:**
   Copy the remote URL from your new GitHub repo and run:
   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   git branch -M main
   git push -u origin main
   ```

## Key Features

- **Cinematic Hero:** Fullscreen animated product showcase.
- **AI Flavor Sommelier:** Personalized recommendations using Genkit.
- **Ecommerce Ready:** Integrated cart system and Razorpay test-mode checkout.
- **Luxury UI:** Built with ShadCN, Tailwind CSS, and Framer Motion for smooth transitions.
